API Standards Reference

API 610 - Centrifugal Pumps for Petroleum, Petrochemical and Natural Gas Industries

1. Scope

  • Specifies requirements for centrifugal pumps in petroleum, petrochemical, and gas industry services
  • Applicable to overhung, between bearings, and vertically suspended pumps
  • Covers pump design, materials, inspection, testing, and preparation for shipment

2. Pump Designations

  • OH1 - Overhung, centerline mounted
  • OH2 - Overhung, centerline mounted, heavy duty
  • BB1 - Between bearings, axially split
  • BB2 - Between bearings, radially split
  • VS1 - Vertically suspended, single casing
  • VS6 - Vertically suspended, can type

3. Key Requirements

  • Operating Temperature Range: -73°C to 425°C (-100°F to 800°F)
  • Minimum Design Pressure: 40 bar (580 psi)
  • NPSH margin: Minimum 10% or 1m (3ft), whichever is greater
  • Bearing Life: Minimum L10h of 25,000 hours
  • Shaft Deflection: Maximum 0.05mm (0.002in) at seal chamber

API 682 - Shaft Sealing Systems for Centrifugal and Rotary Pumps

1. Seal Categories

  • Category 1 - Non-cartridge, elastomer bellows, pusher
  • Category 2 - Cartridge, balanced, standardized
  • Category 3 - Cartridge, high-performance, engineered

2. Seal Types

  • Type A - Pusher seal, dynamic O-ring
  • Type B - Metal bellows seal
  • Type C - High temperature metal bellows seal

3. Seal Arrangements

  • Arrangement 1 - Single seal
  • Arrangement 2 - Dual unpressurized seal
  • Arrangement 3 - Dual pressurized seal

4. Piping Plans

  • Plan 11 - Seal chamber recirculation
  • Plan 21 - Cooled seal chamber recirculation
  • Plan 23 - Cooled seal chamber recirculation with barrier/buffer fluid
  • Plan 52 - Unpressurized dual seal buffer system
  • Plan 53A/B/C - Pressurized dual seal barrier system
  • Plan 54 - Pressurized external barrier fluid system
